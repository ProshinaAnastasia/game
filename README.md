# Snake Game (Игра "Змейка")

Классическая игра "Змейка", реализованная на C++ с использованием библиотеки SFML.

## Используемые паттерны проектирования

В этом проекте использованы следующие паттерны проектирования:

1. **Singleton** - класс `Game` реализует паттерн Singleton для обеспечения единственного экземпляра игры
2. **State** - используется для управления различными состояниями игры (игра, пауза, конец игры)
3. **Command** - используется для обработки ввода пользователя и управления змейкой
4. **Observer** - используется для отслеживания столкновений и поедания пищи

## Требования

- C++17 или выше
- SFML 2.5 или выше
- CMake 3.10 или выше

## Сборка проекта

### Linux/macOS

```bash
# Клонирование репозитория
git clone <repository-url>
cd snake-game

# Сборка проекта
chmod +x build.sh
./build.sh
```

### Windows

```bash
# Клонирование репозитория
git clone <repository-url>
cd snake-game

# Сборка проекта с помощью CMake
mkdir build && cd build
cmake ..
cmake --build .
```

## Управление в игре

- **Стрелки**: Управление змейкой
- **P**: Пауза/возобновление игры
- **R**: Перезапуск игры после проигрыша
- **ESC**: Выход из игры

## Структура проекта

- `include/` - заголовочные файлы
- `src/` - исходные файлы
- `assets/` - ресурсы для игры (шрифты и т.д.)
